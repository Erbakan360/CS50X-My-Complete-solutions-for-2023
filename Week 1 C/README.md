# Week 1: C

**hello.c**: A simple introductory program that prompts the user for their name and prints a personalized greeting, demonstrating basic input/output in C.

**mario.c**: Recreates the iconic pyramids from Super Mario Bros. using nested loops.

**cash.c / credit.c**: Algorithms for greedy change-making (Cash) or validating credit card numbers using Luhn’s Algorithm (Credit).

**Task requirements:** https://cs50.harvard.edu/x/2023/psets/1/

# How to run

1. Open your terminal.
2. Compile the code using: `make filename` (e.g., `make mario`).
3. Execute the program using: `./filename` (e.g., `./mario`).
