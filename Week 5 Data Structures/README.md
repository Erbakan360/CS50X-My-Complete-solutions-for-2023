# Week 5: Data Structures

**speller.c**: A high-speed spell-checker that loads a dictionary into a Hash Table to verify the spelling of every word in a text file.

**Task requirements:** https://cs50.harvard.edu/x/2023/psets/5/

# How to run

1. Open your terminal.
2. Compile the code: `make speller`.
3. Execute the program: `./speller texts/holmes.txt`.
