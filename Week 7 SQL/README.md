# Week 7: SQL

**movies**: A series of SQL queries executed on an IMDb database to answer complex questions about actors and ratings.

**fiftyville**: A data-driven mystery where SQL queries are used on police reports, ATM logs, and flight records to catch a thief.

**Task requirements:** https://cs50.harvard.edu/x/2023/psets/7/

# How to run

1. Open your terminal.
2. Open the database: `sqlite3 movies.db`.
3. Run queries using: `.read 1.sql` or paste the SQL code directly.
