CS50x: Introduction to Computer Science

This repository contains my solutions and projects for the 2023 edition of Harvard University's CS50x. The course covers the fundamental concepts of computer science and programming, transitioning from low-level memory management to high-level web development.

The projects in this repository track my progress through the following topics:
    
    C Programming: Memory management, pointers, and data structures.
    Algorithms: Big O notation, sorting (Bubble, Selection, Merge), and searching.
    Data Structures: Linked lists, hash tables, and tries.
    Python: High-level scripting, automation, and data analysis.
    SQL: Relational database design and complex querying.
    Web Development: Front-end (HTML/CSS/JS) and back-end (Flask).


Course: https://cs50.harvard.edu/x/2023/
