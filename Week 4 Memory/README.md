# Week 4: Memory

**filter.c**: A program that applies various filters to BMP images (grayscale, sepia, reflection, blur) by manipulating individual pixel data.

**recover.c**: A forensic tool that searches through a raw memory image file to reconstruct deleted JPEG files.

**Task requirements:** https://cs50.harvard.edu/x/2023/psets/4/

# How to run

1. Open your terminal.
2. Compile the code: `make filename`.
3. For filter: `./filter -g input.bmp output.bmp`.
4. For recover: `./recover card.raw`.
