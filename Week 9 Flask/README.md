# Week 9: Flask

**birthdays**: A web application that allows users to record and track birthdays in a SQL database.

**finance**: A stock trading platform where users can "buy" and "sell" stocks using live market data from the IEX API.

**Task requirements:** https://cs50.harvard.edu/x/2023/psets/9/

# How to run

1. Open your terminal.
2. Install dependencies: `pip install -r requirements.txt`.
3. Set your API key (for finance).
4. Start the app: `flask run`.
