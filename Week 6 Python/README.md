# Week 6: Python

**mario.py / cash.py / readability.py**: Python implementations of earlier C projects to demonstrate high-level syntax and dynamic typing.

**dna.py**: A forensic tool that identifies individuals by matching DNA Short Tandem Repeats (STRs) against a CSV database.

**Task requirements:** https://cs50.harvard.edu/x/2023/psets/6/

# How to run

1. Open your terminal.
2. Run the script using Python: `python filename.py`.
3. For DNA: `python dna.py databases/large.csv sequences/5.txt`.
