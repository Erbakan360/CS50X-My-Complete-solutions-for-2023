# Week 3: Algorithms

**plurality.c**: A program that simulates a plurality election where the candidate with the most votes wins.

**runoff.c**: Implements a ranked-choice voting system (Instant Runoff) that handles multi-round eliminations.

**tideman.c**: A highly complex election system that guarantees a Condorcet winner by constructing a directed graph of voter preferences.

**Task requirements:** https://cs50.harvard.edu/x/2023/psets/3/

# How to run

1. Open your terminal.
2. Compile the code: `make filename`.
3. Execute the program with candidate names as arguments: `./filename Alice Bob Charlie`.
